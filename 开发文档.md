# 开发文档

## 项目简介

本项目是一个采用 **Swiss Design (瑞士设计风格)** 与 **Raw Minimalism (粗犷极简主义)** 的个人静态主页。拒绝任何花哨的渐变与装饰，强调纯粹的排版、高对比度与信息层级。

## 设计理念 (No AI Flavor)

1. **去装饰化**: 彻底移除所有阴影、圆角、模糊滤镜（Glassmorphism）、极光背景以及动态流光。
2. **高对比单色调**: 仅使用黑、白、灰三种颜色，通过不同明度的对比来区分层级。
3. **侧边栏布局**: 采用固定侧边栏 (Fixed Sidebar) 布局，将导航与主体内容分离，更符合专业工程文档的视觉直觉。
4. **线性元素**: 使用大量的 `1px` 细线作为边界划分，追求工业结实的质感。

## 2025-01-02 深度改版记录

### 2. 精准 OKR 数据建模

- **深度数据对齐**: 按照 2026 规划，精准配置了 PR (50+)、GitHub Contributions (2700+)、漏洞挖掘 (130)、代码行数 (180W) 以及顶会复现 (15) 等核心指标。
- **颗粒度渲染**: 重构了 OKR 渲染引擎，现在能够清晰展示每一个子指标的“当前值 / 目标值”，而不仅仅是概括。
- **履历深度补完**: 详细列出了在 **360 (L2 安全分析工程师)**、**奇安信** 以及 **东软** 的职业轨迹。

### 3. 交互逻辑精简

- **终端 (Terminal)**: 移除复杂的命令交互系统。改为纯文本启动序列显示，仅保留关键的系统自检信息，回归本质。
- **OKR (Progression)**: 进度条改为极细的横向线条，去除所有色彩渐变，仅保留完成度百分比与对应的状态。
- **消除动画焦虑**: 移除了所有滚动入场动画 (`Fade-in`) 和延迟加载，页面加载即呈现，追求极致的响应速度。

### 3. 排版引擎

- **字体**: 使用广受专业设计界认可的 `Inter` 字体。
- **间距**: 采用大面积留白，确保每个板块都有充足的视觉呼吸空间。
