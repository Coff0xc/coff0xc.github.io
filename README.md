# Coff0xc's Personal Homepage

A minimalist, geek-style personal homepage featuring an interactive terminal, CRT scanline effects, and a cyberpunk aesthetic.

## Features

- **Interactive Terminal**: Type `help`, `whoami`, `ls`, `cat`, etc.
- **Visual Effects**: CRT scanlines, glassmorphism, neon accents.
- **Responsive Design**: Works on desktop and mobile.
- **Tech Stack**: HTML5, CSS3, Vanilla JavaScript.

## Deployment

This project is ready to be deployed on [GitHub Pages](https://pages.github.com/).

1. Upload these files to a GitHub repository.
2. Go to **Settings** > **Pages**.
3. Select the `main` branch and `/root` folder.
4. Save.
